 Social-Media-Analytics
 
 Project Overview
This project analyzes social media engagement data to identify user behavior patterns and optimize content strategy.

 Tools Used
- Python (Pandas, NumPy)
- Data Visualization
- Machine Learning (Random Forest)
- Power BI Dashboard

 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Engagement Score Calculation
- Machine Learning Model for Prediction
- Interactive Dashboard

 Key Insights
- Engagement varies significantly across platforms
- Peak engagement occurs during specific hours
- Positive sentiment content performs better
- Shares and impressions drive higher engagement

 Conclusion
This project helps in understanding user engagement and improving social media content strategy using data-driven insights.
